Unity HDRP з»„д»¶зҝ»иҜ‘еӨҮеҝҳ

е®ҳж–№ж–ҮжЎЈ

Volume

  • Is GlobalпјҡеӢҫйҖүеҗҺVolumeеҸҳдёәе…ЁеұҖеҪўејҸпјҢеҪұе“Қж•ҙдёӘеңәжҷҜгҖӮ[1]
  • Blend DistanceпјҡеҪ“Is GlobalжңӘеӢҫйҖүж—¶дјҡжҳҫзӨәжӯӨйЎ№гҖӮеҸҜд»Ҙи®ҫзҪ®дёҖдёӘж··еҗҲеҢәеҹҹпјҢиҜҘеҢәеҹҹдёәд»ҺдёҖдёӘVolumeеҲ°еҪ“еүҚVolumж—¶пјҢеҗ„йЎ№ж•°еҖјжёҗеҸҳзҡ„дёҖдёӘеҢәеҹҹгҖӮ[2]
  • Weightпјҡж··еҗҲзЁӢеәҰпјҢCameraиҝӣе…ҘеҲ°еҪ“еүҚеҢәеҹҹеҗҺпјҢжңҖз»Ҳзҡ„ж··еҗҲз»“жһңгҖӮ[3]
  • PriorityпјҡеҪ“дёӨдёӘVolumeйҮҚеҸ ж—¶пјҢж №жҚ®жӯӨйЎ№еҲӨж–ӯеҪ“еүҚдҪҝз”Ёе“ӘдёҖдёӘVolumeзҡ„й…ҚзҪ®гҖӮиҜҘйЎ№ж•°еҖји¶ҠеӨ§пјҢи¶Ҡдјҡиў«дјҳе…ҲдҪҝз”ЁгҖӮ
  • ProfileпјҡдҪҝз”ЁVolumeй…ҚзҪ®ж–Үд»¶пјҢзӮ№еҮ»вҖңNewвҖқж–°е»әдёҖдёӘй…ҚзҪ®ж–Үд»¶гҖӮ

[1] еҪ“Is GlobalжңӘеӢҫйҖүж—¶пјҢйңҖиҰҒеңЁеҪ“еүҚVolumeжүҖеңЁзү©дҪ“дёҠжҢӮиҪҪBox Colliderз»„д»¶пјҢе№¶дё”еӢҫйҖүBox Colliderз»„д»¶зҡ„Is TriggerгҖӮ
[2] ж··еҗҲеҢәеҹҹжҳҜд»ҺColliderеӨ–дҫ§иҫ№зјҳејҖе§Ӣи®Ўз®—гҖӮдҫӢеҰӮColliderдёә4пјҢBlend Distanceдёә5пјҢж··еҗҲеҢәеҹҹдҪҚзҪ®е°ұжҳҜ4еҲ°9гҖӮ
[3] ж··еҗҲзЁӢеәҰжҳҜ ж‘„еғҸжңәи·қзҰ»е’Ңж··еҗҲи·қзҰ» иҝҗз®—з»“жһңеҗҺпјҢеҶҚд№ҳд»Ҙж··еҗҲзЁӢеәҰж•°еҖјжңҖз»ҲжүҖеҫ—зҡ„з»“жһңгҖӮдҫӢеҰӮпјҢеҪ“еүҚVolumeеҢәеҹҹи®ҫзҪ®дә®еәҰдёә100пјҢж··еҗҲзЁӢеәҰдёә0.5пјҢеңЁиҜҘVolumeдёӯжңҖз»Ҳдә®еәҰдёә50пјҢеҪ“иҝӣе…Ҙж··еҗҲеҢәеҹҹж—¶пјҢжүҖжңүжёҗеҸҳзҡ„ж•°еҖјд№ҹдјҡд№ҳд»Ҙ0.5гҖӮ

е®ҳж–№ж–ҮжЎЈ

Visual Environment

иҜҘжЁЎеқ—дё»иҰҒжҺ§еҲ¶еӨ©з©әе’ҢдёҺйӣҫж•ҲгҖӮ

  • TypeпјҡеӨ©з©әзӣ’зұ»еһӢ
    • Noneпјҡ дёҚдҪҝз”ЁеӨ©з©әзӣ’пјҢCameraеңЁеҪ“еүҚVolumeдёӯзҡ„ж—¶еҖҷпјҢдҪҝз”ЁзәҜиүІгҖӮ
    • Gradient Skyпјҡ дҪҝз”ЁжўҜеәҰеӨ©з©әзӣ’гҖӮ
    • HDRI Skyпјҡ дҪҝз”Ёcubemapзұ»еһӢзҡ„HDRIеӣҫзүҮдҪңдёәеӨ©з©әзӣ’гҖӮ
    • Procedural Skyпјҡ еҸӮж•°иҜқеӨ©з©әзӣ’пјҢеҸҜд»ҘжҺ§еҲ¶ең°йқўйўңиүІпјҢеӨ©з©әиүІи°ғпјҢеӨӘйҳіе°әеҜёзӯүгҖӮ
  • Ambient ModeпјҡзҺҜеўғе…ү(ambient light)зҡ„жЁЎејҸгҖӮ
    • Staticпјҡ зҺҜеўғе…үз…§жқҘиҮӘдәҺзғҳз„ҷзҡ„зҺҜеўғе…ү
    • Dynamicпјҡ зҺҜеўғе…үжқҘиҮӘеҪ“еүҚдҪҝз”Ёзҡ„еӨ©з©әзӣ’гҖӮиҝҷж„Ҹе‘ізқҖпјҢеҪ“еүҚж‘„еғҸжңәжүҖеңЁзҡ„VolumeдёӯпјҢзҺҜеўғе…үеҸҜд»Ҙе®һж—¶еҸҳеҢ–гҖӮеҰӮжһңдҪҝз”Ёзғҳз„ҷе…ЁеұҖз…§жҳҺ(baked global illumination)[2]пјҢж”№еҸҳзҺҜеўғе…үз…§еҸӘдјҡеҪұе“Қиў«зҺҜеўғжҺўй’Ҳ(Ambient Probes)иҰҶзӣ–зҡ„зү©дҪ“[3]гҖӮ
      еҰӮжһңеңЁLighting SettingsйқўжқҝдёӯеӢҫйҖүдәҶе®һж—¶зҺҜеўғе…үз…§(real-time global illumination)пјҢж”№еҸҳзҺҜеўғе…үз…§е°ҶдјҡеҗҢж—¶еҪұе“ҚйқҷжҖҒзү©дҪ“е’ҢзҺҜеўғжҺўй’Ҳ(Ambient Probes)жүҖиҰҶзӣ–зҡ„зү©дҪ“гҖӮ

[1] еҪ“ Ambient Mode йҖүжӢ© Static ж—¶пјҢйңҖиҰҒеңЁеҪ“еүҚзү©дҪ“дёҠж·»еҠ вҖңStatic Lighting SkyвҖқз»„д»¶пјҢиҜҘз»„д»¶еј•з”ЁеҪ“еүҚVolumeдҪҝз”Ёзҡ„й…ҚзҪ®ж–Үд»¶пјҢе№¶жҢҮе®ҡеҪ“еүҚдҪҝз”Ёе“ӘдёӘеӨ©з©әзӣ’гҖӮзғҳз„ҷе®ҢжҲҗеҗҺпјҢйқҷжҖҒжёёжҲҸзү©дҪ“дјҡдҪҝз”Ёзғҳз„ҷж—¶зҡ„зҺҜеўғе…үз…§пјҢжӯӨж—¶зҺҜеўғе…үж”№еҸҳзҡ„иҜқпјҢйқҷжҖҒжёёжҲҸзү©дҪ“дёҚдјҡйҡҸд№ӢеҸҳеҢ–гҖӮ
[2] baked global illuminationжҳҜеңЁLighting SettingsйқўжқҝйқўжқҝдёӯеӢҫйҖүBaked Global IlluminationгҖӮ
[3] Ambient Probesе°ҡдёҚжҳҺзЎ®пјҢUnityж–ҮжЎЈдёӯйҖҡзҜҮжңӘжүҫеҲ°е…ідәҺиҝҷдёӘдёңиҘҝзҡ„иҜҰз»ҶжҸҸиҝ°пјҢеңЁAPIж–ҮжЎЈдёӯд№ҹеҸӘжҳҜеҜҘеҜҘеҮ еҸҘгҖӮдёӘдәәж„ҹи§үеҘҪеғҸжҳҜLight ProbesгҖӮ

PSпјҡжҜҸз§ҚеӨ©з©әзӣ’з»„д»¶еҸҜд»ҘзӢ¬з«Ӣж·»еҠ пјҢж №жҚ®вҖңTypeвҖқеұһжҖ§жқҘзЎ®е®ҡдҪҝз”Ёе“Әз§ҚеӨ©з©әзӣ’гҖӮ

Fog

  • Typeпјҡ йӣҫж•Ҳзұ»еһӢ
    • Noneпјҡ еңЁеҪ“еүҚVolumeдёӯдёҚдҪҝз”Ёд»»дҪ•йӣҫж•ҲгҖӮ
    • Linear Fogпјҡ зәҝжҖ§йӣҫж•ҲпјҢдә§з”ҹзҡ„йӣҫж•Ҳи·қзҰ»Cameraи¶ҠиҝңпјҢеҜҶеәҰдјҡжҲҗзәҝжҖ§еҸҳеӨ§гҖӮ
    • Exponential Fogпјҡ жҢҮж•°еһӢйӣҫж•ҲпјҢдә§з”ҹзҡ„йӣҫж•Ҳи·қзҰ»Cameraи¶ҠиҝңпјҢеҜҶеәҰдјҡжҲҗжҢҮж•°зә§еҸҳеӨ§гҖӮ[1]
    • Volumetric Fogпјҡ дҪ“з§ҜйӣҫпјҢдҪ“з§ҜйӣҫдјҡдёҺзңҹе®һе…үз…§дә§з”ҹдәӨдә’пјҢиғҪдә§з”ҹзңҹе®һзҡ„зү©зҗҶжёІжҹ“ж•ҲжһңгҖӮ[2]

[1] жҢҮж•°йӣҫж•Ҳ е’Ң зәҝжҖ§йӣҫж•ҲзӣёеҗҢзӮ№жҳҜйғҪйҡҸзқҖи·қзҰ»Cameraи¶ҠиҝңеҸҳеҫ—и¶Ҡжө“еҜҶпјҢиҖҢж„ҹи§үжҢҮж•°йӣҫж•ҲдјҡеҸҳеҢ–зҡ„жҜ”зәҝжҖ§йӣҫж•ҲжӣҙеҠ еҝ«гҖӮ
[2] ж„ҹи§үзұ»дјјдҪ“з§Ҝйӣҫзҡ„ж•ҲжһңпјҢе…үзәҝз©ҝиҝҮеҗҺе°ұдјҡжңүдҪ“з§Ҝе…үзҡ„ж•ҲжһңгҖӮ
